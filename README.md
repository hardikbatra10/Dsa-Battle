# DSA Battle

A competitive coding platform where users can:

- Register and Login using JWT Authentication
- Create custom coding rooms
- Join rooms using room codes
- Solve randomly selected DSA problems
- Compete with friends in real-time battles

## Tech Stack

### Backend
- Django
- Django REST Framework
- PostgreSQL
- JWT Authentication

### Features Implemented

- User Registration
- User Login
- User Profile (Me API)
- Problem Creation
- Problem Listing
- Room Creation
- Room Joining

## Future Features

- Code Submission System
- One vs One Battles
- Leaderboards
- Problem of the Day
- WebSocket Real-time Rooms
